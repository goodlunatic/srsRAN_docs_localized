问题反馈
--------

对于代码贡献和问题反馈，我们通常建议用户直接使用 Github 的 `pull request <https://github.com/srsran/srsran/pulls>`_ 或 `issue <https://github.com/srsran/srsran/issues>`_ 跟踪系统。这样便于统一跟踪，也能让公开用户提交的增强和修复立即对所有人可见。

对于可能影响用户安全的 issue 或 fix，我们还提供专用邮箱 ``security@srs.io`` 用于私下沟通。srsRAN 团队会认真检查并分析每一份提交，并分配相应的 threat category，以确保问题被妥善处理。
