srsRAN 文档
===========

srsRAN software suite 是由 `SRS <https://www.srs.io>`_ 提供的一组开源 4G 和 5G software radio 实现。应用使用可移植的 C++ 编写，并尽量减少第三方依赖。基于 AGPLv3 许可，srsRAN 应用可运行在 Linux 上，并使用通用计算与 radio 硬件。

----

**srsRAN Project** 是一个 ORAN-native 的 5G CU/DU，提供从 I/Q 到 IP 的完整协议栈。你可以在 `此处 <https://docs.srsran.com/projects/project/en/latest/>`_ 查看 srsRAN Project 文档。

----

**srsRAN 4G** 套件提供端到端的 4G network 实现，包含 UE、eNodeB 和 EPC 的 full-stack 应用。你可以在 `对应页面 <https://docs.srsran.com/projects/4g/en/latest/>`_ 查看 srsRAN 4G 文档。
