开始使用

要开始使用 srsENB，您需要一台带有基于 GNU/Linux 的操作系统和 SDR RF 前端的 PC。 SDR RF 前端是一种通用无线电设备，例如 Ettus Research USRP，它连接到您的 PC 并支持原始无线电信号的传输和接收。

如果您使用 Ubuntu 操作系统，您可以从提供的二进制包安装 srsENB：

sudo add-apt-repository ppa:srslte/releases
sudo apt-get update
sudo apt-get install srsenb

如果您使用其他发行版，可以参考项目的 GitHub 页面，按源代码方式安装。

安装完成后，您可以运行以下命令，将配置文件安装到默认位置（~/.config/srsran_4g）：

srsran_4g_install_configs.sh user

运行软件

要使用默认参数运行 srsENB，请运行 sudo srsenb 在命令行上。 srsENB 需要以 sudo 管理员权限运行才能创建高优先级线程。启动后，srsENB 将尝试查找您的 RF 前端设备，尝试连接到核心网络 (EPC) 并开始广播。

配置

eNodeb可以通过配置文件进行配置： enb.conf。该配置文件提供与小区配置、工作频率、发射功率级别、记录级别等相关的参数。要使用已安装的配置文件运行 srsENB，请使用 sudo srsenb ~/.config/srsran/enb.conf.

配置文件中指定的所有参数也可以在命令行上覆盖。例如，要使用不同的 EARFCN 运行 eNodeB，请使用 sudo srsenb ~/.config/srsran_4g/enb.conf --rf.dl_earfcn 3350.

除了顶层配置文件之外，srsENB还使用单独的文件来配置SIB（sib.conf）、无线电资源（rr.conf）和数据承载（drb.conf）。这些附加配置文件列在顶级 enb.conf 的 [enb_files] 下，并为每个文件提供了默认值。

eNodeB的一个关键参数是enb.mme_addr，它指定核心网MME的IP地址。默认配置假定 srsEPC 在同一台计算机上运行。有关更多信息以及在单独计算机上使用 EPC 的说明，请参阅 EPC 用户手册。

硬件设置

要使用 srsENB 创建无线本地小区，您将需要 RF 前端和合适的天线。默认 EARFCN 为 3400（上行 2565MHz，下行 2685MHz）。为了减少 TX-RX 串扰，我们建议将 TX 和 RX 天线彼此成 90 度角。

srsENB 也可以通过有线连接使用。电缆配置和所需的 RF 组件将取决于您的 RF 前端。对于 RF 前端（例如 USRP），将 TX 连接到 RX 并确保至少 30dB 的衰减，以避免损坏您的设备。有关有线连接的更多详细信息，请参阅 高级用法.

操作系统设置

srsENB 在具有标准 Linux 内核的用户空间中运行。为了获得最佳性能，我们建议禁用 CPU 频率缩放。要禁用频率缩放，请使用：

echo "performance" | sudo tee /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor

观察结果

要观察 srsENB 结果，请使用生成的日志文件和数据包捕获。

默认情况下，日志文件创建于 /tmp/enb.log。 srsENB 配置文件可用于指定网络堆栈每一层的日志级别并启用十六进制消息输出。支持的日志级别包括调试、信息、警告、错误和无。

日志消息采用以下格式：

Timestamp  [Layer ]  Level    Content

例如：

17:52:25.246 [RLC ]  Info    DRB1 Tx SDU

或者启用十六进制消息输出：

17:52:25.246 [RLC ]  Info    DRB1 Tx SDU
         0000: 8b 45 00 00 c7 f3 8b 40 00 01 11 d1 f6 c0 a8 03
         0010: 01 ef ff ff fa 92 55 07 6c 00 b3 ee 41 4d 2d 53

PHY 层日志消息具有其他详细信息：

Timestamp  [Layer]  Level  [Subframe] Channel:  Content

例如：

17:52:26.094 [PHY1]  Info  [05788]  PDSCH:    l_crb= 1, harq=0, snr=22.1 dB, CW0: tbs=55, mcs=22, rv=0, crc=OK, it=1, dec_time=  12 us
