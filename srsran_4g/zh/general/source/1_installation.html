安装指南

我应该使用哪种安装？

srsRAN 4G 可以从包或源安装。以下决策树应帮助用户决定哪一个最适合他们：

简而言之，寻求简单安装且仅希望通过 USRP 前端运行基本 srsRAN 4G 应用程序的用户应使用软件包安装。 希望修改 srsRAN 4G 和/或使用替代 RF 前端（例如 LimeSDR 和 BladeRF）的用户应从源安装。

包安装

srsRAN 4G 软件套件可以在 Ubuntu 上使用软件包安装：

须藤 添加-适合-存储库 聚苯胺:软件无线电系统公司/斯鲁斯兰
须藤 适合-得到 更新
须藤 适合-得到 安装 斯鲁斯兰 -y

软件包安装也可用于其他发行版。

  • 开放SUSE

  • 拱门 Linux

  • Debian（Pop 操作系统、Mint 等）

请注意，SRS 仅维护 Ubuntu 的 Launchpad 软件包。不同的发行版将维护自己的 srsRAN 4G 软件包，该软件包可能是最新的，也可能不是最新的。安装前检查可用版本 以确保您使用的是所需版本的 srsRAN 4G。

Fedora 尚不支持 srsRAN 4G 的软件包安装。

或者在 Fedora 上：

dnf 安装 cmake fftw3-开发 姆贝德尔斯-开发 陆金所-工具-开发 库配置-开发 提升-开发

请注意，根据您的 Linux 的风格和版本，实际的包名称可能会有所不同。

  • 可选要求：

    • 图形用户界面 - 用于实时绘图。

    • libpcsclite-dev - 用于访问智能卡读卡器

    • libdw-dev libdw - 使用向后cpp 进行真正信息丰富的回溯

  • RF前端驱动：

    • 超高清

    • 肥皂SDR

    • 刀片射频

    • 零MQ

注意事项

如果使用 UHD，我们推荐 UHD 的 LTS 版本，即 3.9.7 或 3.15。

警告

必须安装所有强制要求、可选要求和 RF 前端驱动程序 先前的 到建筑 srsRAN 4G。如果不这样做将导致 运行时错误或阻止 srsRAN 4G 正确构建。
