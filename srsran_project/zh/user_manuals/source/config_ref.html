配置参考

srsRAN Project 使用 YAML (.yml) 配置文件。

gNB 附带了许多示例配置文件，这些可以在 配置/ 源文件中的文件夹。

这些配置文件示例提供了让用户启动和运行的基本设置，用户可以轻松修改这些设置以适合他们的用例。

其中包括示例补充配置文件：

  • MIMO 与 USRP

  • QAM 从 64 增加到 256

  • 针对语音、视频、IMS、直播和缓冲视频流的特定 QOS 配置

  • 切片支持

srsRAN Project 允许配置文件与其他配置文件串联使用，使用 -c 配置文件之间。例如，您可能想要同时使用 n310 和 MIMO 示例配置，如下所示：

sudo ./gnb -c gnb_rf_n310_fdd_n3_20mhz.yml -c mimo.yml

此外，除了上述之外，您可能还希望使用 QAM256。这可以按如下方式完成：

sudo ./gnb -c gnb_rf_n310_fdd_n3_20mhz.yml -c mimo.yml -c qam256.yml

天线配置

srsRAN Project 支持 SISO、基于码本的 MIMO 和 TX/RX 分集。这些选项可以在 单元格配置 部分通过 nof_天线_dl 和 nof_天线_ul 参数。

当您配置时 nof_天线_dl 和 nof_天线_ul，它分别通知 gNB 可用于下行链路 (DL) 和上行链路 (UL) 传输的天线数量。默认情况下，一个 DL 和 UL 使用单个物理天线，这意味着 nof_天线_dl 和 nof_天线_ul 不是添加剂。
